Visual Tab Switcher

Fast tab switching with visual previews.

Shortcuts

Open Tab Switcher Alt + Q
Quick Cycle Ctrl + Shift + K
Tip: Want Ctrl+Tab? Visit chrome://extensions/shortcuts

Controls

  • ↑ ↓ ← → Navigate
  • Enter Switch tab
  • Delete Close tab
  • . Recently closed
  • Esc Close overlay
  • Type to search

Features

  • Visual thumbnail previews
  • Instant overlay (<100ms)
  • Smart screenshot caching
  • Recent tab sorting
  • Fuzzy search
  • Light & dark theme
